# CLI Reference

## Usage

```bash
bash scripts/creator.sh <command>
```

---

## Commands

### help

Display available commands.

```bash
bash scripts/creator.sh help
```

---

### doctor

Run workstation diagnostics.

```bash
bash scripts/creator.sh doctor
```

---

### status

Display workstation status.

```bash
bash scripts/creator.sh status
```

---

### cleanup

Run cleanup utilities.

```bash
bash scripts/creator.sh cleanup
```

---

### backup

Run backup tasks.

```bash
bash scripts/creator.sh backup
```

---

### update

Update workstation.

```bash
bash scripts/creator.sh update
```

---

### install

Install workstation components.

```bash
bash scripts/creator.sh install
```

---

### provision

Execute provisioning framework.

```bash
bash scripts/creator.sh provision
```

---

### config

Manage configuration.

```bash
bash scripts/creator.sh config
```

---

### info

Display workstation information.

```bash
bash scripts/creator.sh info
```

---

### version

Display CLI version.

```bash
bash scripts/creator.sh version
```
