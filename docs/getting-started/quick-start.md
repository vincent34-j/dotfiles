# Quick Start

After installation, Creator Workstation is ready to use.

---

# Display Help

```bash
bash scripts/creator.sh
```

---

# Show Status

```bash
bash scripts/creator.sh status
```

---

# Run Diagnostics

```bash
bash scripts/creator.sh doctor
```

---

# Run Tests

```bash
bash scripts/tests/run.sh
```

---

# Update

```bash
bash scripts/creator.sh update
```

---

# Backup

```bash
bash scripts/creator.sh backup
```

---

# Cleanup

```bash
bash scripts/creator.sh cleanup
```
