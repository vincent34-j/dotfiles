# Creator Workstation Dotfiles

This repository contains the configuration, scripts, and documentation for my Linux Creator Workstation.

## Goals

- Reproducible Linux Mint workstation
- Git version-controlled configuration
- Automated software installation
- Automated backup
- Development environment
- AI workspace
- Creator workflow

## Repository Structure

- config/     Configuration files
- docs/       Documentation
- scripts/    Utility scripts

This repository is the single source of truth for rebuilding my workstation.
